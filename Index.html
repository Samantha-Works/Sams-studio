⚙️ How to enable photo processing locally

Restore / Enhance (GFPGAN + Real-ESRGAN):

pip install basicsr facexlib realesrgan
pip install gfpgan

Colorize (DeOldify / DDColor):

pip install deoldify
# Or: git clone https://github.com/nicehuster/DDColor && pip install -r requirements.txt

Animate (SadTalker / LivePortrait):

git clone https://github.com/Winfredy/SadTalker
cd SadTalker && pip install -r requirements.txt
# Download checkpoints: bash scripts/download_models.sh

Photo→Video (Stable Video Diffusion or AnimateDiff):

pip install diffusers transformers accelerate torch
# Model: stabilityai/stable-video-diffusion-img2vid (~25GB)
